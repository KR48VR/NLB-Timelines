      Sources and caveats

      How the data was assembled

      Current public-library names, addresses, coordinates, and most branch histories come from the National Library Board library listing API and branch pages. Historical national-library lineage comes from NLB's official History of National Library page. National Archives of Singapore and Former Ford Factory are included as a separate archives/heritage layer because NLB lists NAS under its current institutions and locations. OneMap address search was used to place former sites where current NLB coordinates no longer apply. A few exact dates for former/specialised libraries that were not present in the current NLB API are marked as secondary cross-checks. The old Stamford Road marker is approximate to the former 91 Stamford Road site.
